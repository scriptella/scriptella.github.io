Scriptella ETL Project Support

Scriptella is maintained through focused compatibility and bug-fix releases. Release 1.3 RC1 is the current source baseline: Java 8, Maven module builds, and the Ant-built standalone distributions. Official final artifacts have not yet been published; Scriptella 1.2 remains the latest generally available release.

Community support

For community help, use GitHub Discussions or ask on Stack Overflow. The historical Scriptella Google Group is no longer active.

Commercial support

For business use, commercial support options, or other inquiries, email scriptella@gmail.com. Demand for paid support and larger feature work is evaluated as it arises; the open-source project itself focuses on maintenance and compatibility.

Scriptella is maintained by PVR Labs — Java systems engineering and developer tools.

Issue tracker

Report bugs and propose focused improvements in the GitHub issue tracker.
